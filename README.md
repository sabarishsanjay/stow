# STOW Updates

This repository hosts the Sparkle appcast for STOW automatic updates.

## Files

- `appcast.xml` - Sparkle update feed containing version information and download links

## About STOW

STOW is a fast video downloader for macOS that supports YouTube, Twitter, Instagram, TikTok, and 1000+ other sites.
